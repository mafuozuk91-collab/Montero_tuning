Инклиометр (кренометр) на базе Arduino, MPU6050, Oled 128x64.  
Климат контроль передней и задней печки на базе ArduinoMega_2560, TFT_420x380, lcd1602.(все еще в разработке)  
Мини бортовой компьютер вместо стандартных часов: часы, дата, инклиометр, управление пневмобалонами в задних пружинах.(все еще в рвзрвботке)
